 STEP 2: OPEN POWERSHELL/TERMINAL
================================================================================

Navigate to backend folder:

    cd d:/TounsiVert/backend


================================================================================

================================================================================

    php artisan migrate


You should see:

================================================================================

 STEP 5: CREATE STORAGE LINK

================================================================================
 üêõ TROUBLESHOOTING
================================================================================

‚ùå ERROR: "SQLSTATE[HY000] [1045] Access denied"
    FIX: Update database password in .env file
    Open: d:/TounsiVert/backend/.env
    Change: DB_PASSWORD= (to your MySQL password)
    Then run: php artisan migrate again

‚ùå ERROR: "Base table or view not found"
    FIX: Run migrations again
    php artisan migrate:fresh --seed

‚ùå ERROR: "Class 'OrgCategorySeeder' not found"
    FIX: Run this, then seed again
    composer dump-autoload
    php artisan db:seed

‚ùå ERROR: "Vite manifest not found"
    FIX: Make sure npm run dev is running

‚ùå ERROR: "Permission denied" on Windows
    FIX: Run PowerShell as Administrator, then:
    icacls storage /grant "Users:(OI)(CI)F" /T
    icacls bootstrap/cache /grant "Users:(OI)(CI)F" /T
